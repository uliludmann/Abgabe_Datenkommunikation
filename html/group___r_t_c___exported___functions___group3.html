void HAL_RTC_AlarmAEventCallback ( RTC_HandleTypeDef *  hrtc)

ALARM A Event Callback in non blocking mode.

Note
This function is called when RTC_ALARM interrupt took place, inside RTC_IRQHandler(). It makes a direct call to HAL_IncTick() to increment a global variable "uwTick" used as application time base.
Parameters
hrtc: RTC handle
